## Format used for O notation
O(1)
O(n)
O(n!)
n square -> O(n^2)
log(n) -> O(log(n))
n * log(n) -> O(nlog(n))
n + k -> O(n+k)

The "short" notation (don't use constants). Example: O(nk) or O(wn) should be written O(n)
